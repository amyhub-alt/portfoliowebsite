Developed a full-stack movie application using the MERN stack. Developed a Pokémon browsing app using JavaScript and Bootstrap. The app fetches data from the PokéAPI and displays a list of Pokémon with their names and images. The app features a clean and responsive UI, utilizing flexbox for layout and Bootstrap for styling.
